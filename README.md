# Taller LLM - Desmontando Transformers

**Universidad EAFIT** - Prof. Jorge Ivan Padilla Buritica

## Descripción

Aplicación interactiva que demuestra:
1. Tokenización de texto (IDs y visualización)
2. Geometría de embeddings con PCA
3. Inferencia con Groq API (temperatura, top-p)
4. Métricas de rendimiento (time/token, throughput)

## Despliegue online

https://taller-llm-groq.streamlit.app

```bash
pip install -r requirements.txt
